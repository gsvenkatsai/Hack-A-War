04   Architecture

AWS-Powered
Debt Propagation Engine

Input + Compute Layer
  • Developer pastes GitHub URL into dashboard
  • AWS Amplify hosts the React frontend
  • API Gateway routes the incoming request
  • Lambda 1: repo clone + static analysis (Radon, Lizard, pip-audit, coverage.py)
  • Lambda 2: dependency graph via Python ast module
  • Lambda 3: blast radius calculation per module
  • Lambda 4: LLM enrichment + fix simulation calls
Storage + LLM Layer
  • DynamoDB: stores findings, debt scores, graph data, simulation results
  • S3: raw repo files, generated PDF reports, CTO one-pagers
  • Groq API + Llama 3.1 70B: 4 sequential LLM calls
  • Fingerprint → Enrich → Simulate → Plan
  • Free tier — no AWS Bedrock credits required
  • Amazon Q Developer: code quality and security scanning
